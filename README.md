# 📈 Algorithmic Alpha: Evaluating Risk-Adjusted Returns

## Overview

This project analyzes stock and portfolio performance using Python and quantitative portfolio analytics techniques. The objective is to evaluate investment performance using risk-adjusted measures and visualize portfolio returns.

---

## Project Objectives

- Analyze historical stock price data
- Calculate daily and cumulative portfolio returns
- Measure portfolio risk using standard deviation
- Evaluate investment performance using the Sharpe Ratio
- Visualize portfolio performance using charts
